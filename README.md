

Apple Landing Page Clone

Description

This is a frontend clone of the Apple landing page hero section which focuses on HTML and CSS skills, including layout, typography, buttons, and responsive design.
This project is static, with no JavaScript functionality.


---

Features

Semantic HTML5 structure (header, nav, main, section, footer)

Reusable CSS classes for hero cards

Flexbox layout for nav and hero content

Scalable and maintainable CSS

Responsive design (planned via media queries)

Styled hero section with cards for iPhone, MacBook, and iPad

Buttons styled to match Apple’s look



---

Technologies Used

HTML5

CSS3

Flexbox & Grid (optional for future enhancements)
